Environmental Monitoring Station


Sensor
DHT11 (simulated using DHT22 in Wokwi)

Library
Adafruit DHT Sensor Library

Library Version
1.4.6

Output Format:
timestamp,temp_C,temp_F,humidity

Update Interval:
2 seconds

Note: Wokwi provides a DHT22 component, so the simulation was performed using DHT22. The program logic is identical to the DHT11 version, with only the sensor type definition changed for simulation.
